# All Operations Together

Let's use all the math operations in one program!

```python
a = 10
b = 5

print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division
```

You'll see:
```
15
5
50
2.0
```

You're a math wizard! You can do any calculation!
