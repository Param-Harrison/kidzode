# Final Challenge: Tip Calculator

Build a real tip calculator for restaurants!

Calculate the tip and total bill:
- Bill amount
- Tip percentage (15%, 18%, or 20%)
- Show tip amount
- Show total

Example output:
```
Bill: $50.0
Tip (18%): $9.0
Total: $59.0
```

Make it YOUR OWN! Use any bill amount and tip percentage!
