# Remove Item

Changed your mind? Put it back on the shelf!

Use `.remove()` to take something out.

```python
cart = ["Apples", "Milk", "Broccoli"]
print(cart)

cart.remove("Broccoli")
print(cart)
```

You'll see:
```
['Apples', 'Milk', 'Broccoli']
['Apples', 'Milk']
```
