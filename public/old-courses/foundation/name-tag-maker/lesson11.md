# Format Name Tag with F-Strings

Create a perfect name tag using f-strings!

```python
name = "Jack"
age = 10
border = "=" * 20

print(border)
print(f"   Name: {name}")
print(f"   Age: {age}")
print(border)
```

You'll see:
```
====================
   Name: Jack
   Age: 10
====================
```

A professional name tag! Clean and formatted!

**Why f-strings rock:**
- Easier to read
- Less typing
- Cleaner code
