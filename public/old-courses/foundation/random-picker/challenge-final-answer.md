# Solution

```python
import random

friends = ["Jack", "Sarah", "Mike", "Amy", "Tom", "Zoe"]

captain_a = random.choice(friends)
captain_b = random.choice(friends)

print("--- TEAM GENERATOR ---")
print(f"Team A Captain: {captain_a}")
print(f"Team B Captain: {captain_b}")
```

Output:
```
--- TEAM GENERATOR ---
Team A Captain: Sarah
Team B Captain: Tom
```
