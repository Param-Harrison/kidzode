# Percentage Used

How much of your budget is gone?

Formula: `(spent / budget) * 100`

```python
budget = 100
spent = 25

percent = (spent / budget) * 100
print(f"You used {percent}% of your budget.")
```

You'll see:
```
You used 25.0% of your budget.
```

Keep it low to save more!
