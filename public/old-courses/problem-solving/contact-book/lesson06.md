# Add New Contact

Met someone new? Add them!

Use `.append()` to add a dictionary to the list.

```python
contacts = []

# Add Mario
contacts.append({"name": "Mario", "phone": "123"})

# Add Luigi
new_friend = {"name": "Luigi", "phone": "456"}
contacts.append(new_friend)

print(contacts)
```

You'll see 2 contacts in the list!
