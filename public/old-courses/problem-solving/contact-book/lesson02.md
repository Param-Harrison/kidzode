# More Info

Contacts have more than just phones!
Email, City, Birthday...

```python
contact = {
    "name": "Princess Peach",
    "phone": "555-9999",
    "email": "peach@castle.com",
    "city": "Mushroom Kingdom"
}

print(contact)
```

You can store as much info as you want!
