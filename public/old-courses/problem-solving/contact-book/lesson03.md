# Get Info

Need to call someone? Get their number!

```python
contact = {
    "name": "Toad",
    "phone": "555-1111"
}

print(f"Calling {contact['name']}...")
print(f"Dialing {contact['phone']}...")
```

You'll see:
```
Calling Toad...
Dialing 555-1111...
```

We use the **Keys** (`name`, `phone`) to get the **Values**.
