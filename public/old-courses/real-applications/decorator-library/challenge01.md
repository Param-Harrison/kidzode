# Mini-Challenge 1: Power Factory

Create a function factory called `power_maker(n)`.
It returns a function that raises numbers to the power of `n`.

**Your Goal:**
1. Create `square_it` using `power_maker(2)`.
2. Create `cube_it` using `power_maker(3)`.
3. Test `square_it(4)` -> 16.
4. Test `cube_it(2)` -> 8.

Go! ⚡️
