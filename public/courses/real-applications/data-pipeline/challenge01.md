# Mini-Challenge 1: DIY Range

The built-in `range()` is actually a generator!
Let's build our own calling `my_range(start, end)`.

**Your Goal:**
1. Define `my_range(start, end)`.
2. Use a `while` loop to iterate from `start` to `end` (exclusive).
3. `yield` the current number.
4. Test it in a for loop: `for i in my_range(1, 4): print(i)`
