# Mini-Challenge 2: CLI Calculator

Build a calculator that takes a command and flags.

**Usage Examples:**
- `calc.py add 5 5` -> 10
- `calc.py sub 10 5` -> 5
- `calc.py add 5 5 --loud` -> "THE SUM IS 10!!!"

**Your Goal:**
1. Handle `add` and `sub`.
2. Parse two numbers.
3. If `--loud` is present, print in UPPERCASE with `!!!`.
