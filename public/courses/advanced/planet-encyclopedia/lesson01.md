A list is great for storing names, like `["Mars", "Venus"]`.
But what if we want to store the **size** and **color** too?

### The Solution: Dictionaries
A dictionary uses `{}` and lets us label our data.

```python
mars = {
  "name": "Mars",
  "color": "Red",
  "size": 6779
}
```
