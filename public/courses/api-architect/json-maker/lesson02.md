JSON stores data in **Keys** and **Values**.
Keys are always strings (text). Values can be anything!

### Types Supported
- String: `"name": "Alex"`
- Number: `"age": 10`
- Boolean: `"active": True`

### Your Goal
1. Create a `settings` dictionary.
2. Add a `String` key for "theme".
3. Add a `Number` key for "volume".
4. Add a `Boolean` key for "wifi".
5. Print keys only using `.keys()`.
