# Adding Items (Create) ➕

APIs let us **Create** new data.
In Python lists, we use `.append()`.

### The Logic
```python
new_item = {"name": "Grapes"}
shop.append(new_item)
```

### Your Goal
1. Create an empty `cart = []`.
2. Create `item = {"toy": "car"}`.
3. Append `item` to `cart`.
4. Print the cart.

### Achievement
➕ **Content Creator**: You added data to the system!
