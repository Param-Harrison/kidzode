# Calculate Percentage

Did you get an A? Let's calculate the percentage!

Formula: `(score / total) * 100`

```python
score = 4
total = 5

percent = (score / total) * 100
print(f"You got {percent}%")
```

You'll see:
```
You got 80.0%
```

Math is useful!
