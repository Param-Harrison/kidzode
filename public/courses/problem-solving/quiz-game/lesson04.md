# Question and Answer

For a quiz, we need a Question and an Answer.
Perfect for a dictionary!

```python
qa = {
    "question": "What is the capital of France?",
    "answer": "Paris"
}

print(qa["question"])
```

You'll see:
```
What is the capital of France?
```

We stored a trivia fact!
