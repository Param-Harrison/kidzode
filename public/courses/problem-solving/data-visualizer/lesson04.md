# Calculate Range

How big is the difference?
Range = Max - Min

```python
prices = [10, 50, 5, 100]

high = max(prices)
low = min(prices)
diff = high - low

print(f"Price Range: ${diff}")
```

You'll see:
```
Price Range: $95
```

That's a big difference!
