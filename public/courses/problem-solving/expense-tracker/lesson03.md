# Add Expense

Bought something new? Add it to the list!

```python
expenses = []

# Buying things...
expenses.append({"item": "Candy", "cost": 2.00})
expenses.append({"item": "Soda", "cost": 1.50})

print(expenses)
```

You'll see your shopping list grow!
