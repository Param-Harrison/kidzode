# Biggest Expense

What cost the most?

Use `max()` to find the biggest number!

```python
costs = [10, 50, 5]
biggest = max(costs)

print(f"Most expensive item cost: ${biggest}")
```

You'll see:
```
Most expensive item cost: $50
```

**Pro Tip:** `min()` finds the smallest!
