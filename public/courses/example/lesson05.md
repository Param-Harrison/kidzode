# Lesson 5: If Statements

Learn how to make decisions in your code!

## What you'll learn
- How to use `if`, `elif`, and `else`
- Comparison operators (`>`, `<`, `==`, `!=`)
- Boolean logic

## Instructions
Write a program that:
1. Asks the user for their age
2. If they are 18 or older, print "You are an adult"
3. Otherwise, print "You are a minor"

## Example
```python
age = int(input("Enter your age: "))
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
```

## Your Task
Complete the code to check if someone is an adult.
