# Add and Remove from Lists

Change lists by adding and removing items!

## What to Do
1. Press Run
2. See lists change!

```python
# Start with an empty list
my_list = []

# Add items
my_list.append("apple")
my_list.append("banana")
my_list.append("cherry")
print(f"After adding: {my_list}")

# Get items by position
print(f"First item: {my_list[0]}")

# How many items?
print(f"List has {len(my_list)} items")

# Remove an item
my_list.remove("banana")
print(f"After removing: {my_list}")
```

## Remember
- `.append(item)` adds to the end
- `.remove(item)` removes the item
- `len(list)` tells how many items
