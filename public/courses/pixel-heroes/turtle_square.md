## Let's Do It!
1. Read the code below
2. Press the green **Run** button
3. Watch the turtle draw!

```python
import turtle

# Create a turtle
t = turtle.Turtle()

# Draw a square (4 sides)
t.forward(100)  # Move forward 100 steps
t.left(90)      # Turn left 90 degrees
t.forward(100)
t.left(90)
t.forward(100)
t.left(90)
t.forward(100)
t.left(90)

# Keep the window open
turtle.done()
```

## ğŸ’¡ Important Things to Remember

- `left(90)` turns the turtle left 90 degrees
- A square has 4 equal sides and 4 right angles (90 degrees)
