# Make a Magic 8-Ball

Create a fortune teller that gives random answers!

## What to Do
1. Press Run
2. Run it again - get different fortunes!

```python
import random

# List of fortunes
fortunes = ["Yes, definitely!", "No way!", "Maybe later!"]

# Pick one randomly
answer = random.choice(fortunes)

print("The Magic 8-Ball says...")
print(answer)
```

## Remember
- `import random` gives random powers
- Lists use `[]` and commas
- `random.choice()` picks randomly
