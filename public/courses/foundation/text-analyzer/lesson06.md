# Count Words

Now we can count the words!

Combine `.split()` and `len()`!

```python
sentence = "Python is super cool"
words = sentence.split()
count = len(words)

print(f"Word count: {count}")
```

You'll see:
```
Word count: 4
```

You built a Word Counter! Writers use this all the time!
