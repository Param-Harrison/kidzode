# Solution

```python
fruits = ["Apple", "Banana", "Orange"]
veggies = ["Carrot", "Corn", "Potato"]

print("FRUITS:")
for item in fruits:
    print(f"- {item}")

print("-" * 10)

print("VEGGIES:")
for item in veggies:
    print(f"- {item}")
```

Output:
```
FRUITS:
- Apple
- Banana
- Orange
----------
VEGGIES:
- Carrot
- Corn
- Potato
```
