## Ways to Join Text
You've seen these before:
1. Using `+`: `power + " " + hero`
2. Using commas in print: `print(power, hero)`

There's one more cool way: **f-strings**!

## What's an f-string?
An f-string lets you put variables INSIDE a piece of text:
```python
name = f"{power} {hero}"
```

The `f` before the quotes means "fill in the blanks"!
The `{power}` and `{hero}` are the blanks â€” Python fills them with the actual values.
