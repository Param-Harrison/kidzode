# Money Calculations

Let's calculate money like a real cash register!

```python
item1 = 5.99
item2 = 3.50
item3 = 2.25

total = item1 + item2 + item3
print(f"Total: ${total}")
```

You'll see:
```
Total: $11.74
```

You made a shopping calculator! Using f-strings makes it look professional!
