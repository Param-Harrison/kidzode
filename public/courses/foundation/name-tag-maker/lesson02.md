## Your Mission
We'll use a **variable** (remember ‚Äî it's like a labeled box!) to store your name.

## How It Works
```python
my_name = "Alex"
```

Now `my_name` is a box with "Alex" inside. We can use this box anywhere in our code!

## Printing Variables
We can print text and variables together:
```python
print("*", my_name, "*")
```

The commas separate different things to print. Python automatically adds spaces between them.
