State is what makes your pet unique.
Is it hungry? Is it happy?
We call these "Attributes".

### The Code
```python
class Pet:
    energy = 100
    hunger = 0

buddy = Pet()
buddy.energy = 50 
```

### Practical Use
In games, "Health Points" (HP) and "Mana" are just attributes on the Player object.
