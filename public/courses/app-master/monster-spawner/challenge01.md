# CHALLENGE 1: Custom Names

Give each enemy a unique name like "Orc 1", "Orc 2".
Use the loop variable `i`.

### Your Task:
1. `class Ghost` with `__init__(self, name)`.
2. Loop 5 times.
3. Name should be "Ghost " + str(i).
4. Create Ghost and print name.

### Practical Use
This helps debuggers identify exactly which object is causing a crash.
