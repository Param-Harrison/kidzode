# CHALLENGE 3: Shuffle Mode

Randomize the order!
Use `random.shuffle()`.

### Your Task:
1. `import random`.
2. `class Playlist` with `shuffle()`.
3. Calls `random.shuffle(self.songs)`.
4. Add 3 numbers [1, 2, 3]. Shuffle. Print list.

### Practical Use
"Shuffle" is one of the most strictly algorithmic features in music players.
