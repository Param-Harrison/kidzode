# Lesson 4: Working with Strings

Explore the power of text manipulation!

## What you'll learn
- String methods like `.upper()`, `.lower()`, `.title()`
- String formatting
- String concatenation

## Example Code
```python
name = "alice"
print(name.upper())  # ALICE
print(name.title())  # Alice
print(name.capitalize())  # Alice

message = f"Hello, {name}!"
print(message)
```
