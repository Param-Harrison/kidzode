# CHALLENGE 3: Stopwatch ⏱️

Can you build a simple stopwatch using code?

### Your Task:
1. Import `time`.
2. Ask the user to "Press ENTER to start".
3. Record the `start_time = time.time()`.
4. Ask the user to "Press ENTER to stop".
5. Record the `end_time = time.time()`.
6. Calculate the difference and print: "Time: X seconds".

### Achievement:
You've engineered a precise measuring tool! ⏱️
