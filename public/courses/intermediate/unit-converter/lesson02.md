## Length Converters
```python
def cm_to_inches(cm):
    return cm / 2.54

def meters_to_feet(m):
    return m * 3.281
```

## Weight Converters
```python
def kg_to_pounds(kg):
    return kg * 2.205

def pounds_to_kg(lb):
    return lb / 2.205
```
