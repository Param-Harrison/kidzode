## What's a Function?
A **function** is a reusable piece of code with a name. Think of it like a recipe card â€” you write it once, then use it whenever you need it!

## How to Create a Function
```python
def show_menu():
    print("1. Add Task")
    print("2. View Tasks")
    print("3. Exit")
```

**Breaking it down:**
- `def` = "define" (we're creating something new)
- `show_menu` = the name of our function
- `()` = where info can go in (we'll learn this later!)
- `:` = start the recipe

## How to Use It
Just call its name with `()`:
```python
show_menu()
```

Now all those print lines run! And you can call it again and again.
